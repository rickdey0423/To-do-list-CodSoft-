# 🌟 Vibrant To-Do List 

A sleek, high-performance To-Do application built with a **C++ logic architecture** and a **Modern Web Frontend**.

---

### 🚀 [Click Here for Live Preview](https://rickdey0423.github.io/To-do-list-CodSoft-/)

---

## ✨ Features
- **📋 View tasks in a table format**: Organized and easy to read.
- **✅ Status Tracking**: Mark tasks as Completed or Pending.
- **🔄 Toggle Status**: Quick updates with a single click.
- **🗑 Delete tasks**: Remove completed or unnecessary items.
- **🎨 Eye-soothing UI**: Vibrant gradients and smooth animations.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Vibrant Gradients), JavaScript (ES6)
- **Backend Concept:** C++ (Vector-based data management)

## 📂 Project Structure
- `index.html`: The main user interface and application logic.
- `main.cpp`: Beginner-level C++ code showing the data structure.
- `README.md`: Project documentation and setup guide.
